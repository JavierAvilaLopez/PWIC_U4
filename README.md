# PWIC - U4 - Galeria de Perros

## Estructura de carpetas

- `css/` → Hojas de estilo.
- `js/` → Scripts JS.
- `libs/` → Librerías externas (jQuery local).
- `img/` → Imágenes locales (opcional).
- `index.html` → Landing Page.
- `search.html` → Página de búsqueda.

## Librerías

- jQuery (versión local en `/libs/jquery.min.js`).
